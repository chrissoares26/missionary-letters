**Issues Encountered & Solutions**

1. **Pinia Colada mutation status**
   - Problem: `status.value === 'pending'` is true on initial state ‚Äî buttons disabled on page load
   - Fix: Use `isLoading.value` (true only while actively running)

2. **Edge Function CORS**
   - Problem: CORS headers only on OPTIONS preflight, not on POST responses
   - Fix: Added `corsHeaders` constant spread into ALL response objects (v2)

3. **Inter-function auth (style_match 401)**
   - Problem: `draft_generate` called `style_match` via `adminClient.functions.invoke()` which attaches service role key ‚Äî `style_match`'s `auth.getUser()` can't resolve a user from it
   - Fix: Forwarded original user JWT via `headers: { Authorization: authHeader }` in functions.invoke (v3)

4. **Wrong parameter name**
   - Problem: Sent `top_k: 6` to `style_match` which expects `k`
   - Fix: Changed to `k: 6` (v3)

5. **Realtime not updating UI**
   - Problem: `campaign_content` table not in `supabase_realtime` publication
   - Fix: `ALTER PUBLICATION supabase_realtime ADD TABLE campaign_content;` via migration

6. **CSS variables undefined across all campaign components**
   - Problem: Components used `--primary`, `--border`, `--background`, `--background-secondary`, `--primary-light` ‚Äî none defined in `tokens.css`
   - Actual token names: `--action-primary`, `--border-default`, `--bg-canvas`, `--bg-muted`, `--bg-surface`
   - Fix: Global replace across all 6 affected files
